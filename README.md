# Covid-19_Trend_Analysis_Using_Knime

## Project Overview

This project analyzes Covid-19 trends using the KNIME Analytics Platform and JHU CSSE dataset. The goal is to study confirmed cases, recovered cases, deaths, active cases, and death rate using data analysis and visualization.

## Dataset Used

Source: Johns Hopkins University (JHU CSSE)

Files Used:
- Confirmed Cases
- Death Cases
- Recovered Cases

## Tools Used

- KNIME Analytics Platform  
- JHU CSSE Covid-19 Dataset  

## Visualizations

- Bar Chart (Top Countries)  
- Scatter Plot (Death Rate vs Confirmed Cases)  
- Heatmap (Correlation Matrix)  

## Key Insights

- Covid-19 cases increased rapidly during peak waves  
- Strong correlation between confirmed and recovered cases  
- Death rate varies across countries  
- Top countries contributed major share of cases  

## How to Run

1. Open KNIME Analytics Platform  
2. Import workflow  
3. Load dataset files  
4. Execute workflow
